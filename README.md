# ⚽ 長岡京SS 配車交通費精算アプリ

少年サッカーチーム「長岡京SS」の配車（保護者の車出し）にかかる交通費を、
会場を入れるだけで自動計算するWebアプリ。

**アプリURL：https://kosuking-netizen.github.io/nagaokakyo-ss-carpool/**

- インストール不要。スマホ・PCのブラウザで開くだけ
- 運用コスト **0円**（GitHub Pages・GitHub Actions・NAVITIME API無料枠のみで構成）
- 管理者の定常作業 **ゼロ**（ガソリン価格は毎週自動更新）

---

## 使い方（保護者向け）

1. **① 会場を入れる**：会場名で検索 → 候補をタップ
   → 集合場所（ローソン長岡京インター店）からの最新ルート・高速料金を自動取得
2. **② 確認して計算**：割り勘人数を入れる → 「精算結果をコピー（LINE用）」→ LINEに貼り付け

会場名で見つからないとき（小さな施設は名前検索に出ないことがある）：
- **住所で検索**（例：「田原本町法貴寺」）
- **座標を貼り付け**:Googleマップで地点を長押し → 座標（例 `34.558, 135.796`）をコピーして検索欄へ

2回目以降は「最近の会場」チップをタップするだけ（毎回最新ルートを取り直します）。

## 精算ルール

```
ガソリン代 = 往復距離 ÷ 燃費7km/L × (京都府の週次ガソリン価格 + 10円)  …1円単位に四捨五入
高速代     = ETC片道料金 × 2（NAVITIME実額・ETC普通車）
精算額     = ガソリン代 + 高速代  …10円単位に四捨五入
一人あたり = 精算額 ÷ 割り勘人数  …10円単位に切り上げ（2人以上のとき表示）
```

- ルートは集合場所→会場の**時間最短（有料道路優先）**。計算のたびに取得するため、道路開通・料金改定に自動追従
- **ETC料金は「試合の日付」の朝8時出発として計算**（`start_time`パラメータ）。土日祝の**ETC休日割引が自動反映**される。過去日でも割引判定される。8時固定なのは時刻必須仕様のためで、0〜4時を渡すと深夜割引が誤適用されるので昼間の時刻に固定。日付変更時はルートを自動で取り直す
- ガソリン価格は資源エネルギー庁の週次調査（京都府平均・レギュラー）を毎週水〜金の夜に自動取得
- **ガソリン単価は「計算した時点の最新週の価格」を使う割り切り仕様**（試合直後の精算を想定。週次の価格変動は通常±1〜2円/Lで、精算額への影響は数十円未満のため過去週への遡及適用はしない）
- LINEコピー文に単価・距離・燃費を含むため、受け取った人も検算可能

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（画面・検索・NAVITIME連携。単一ファイル） |
| `calc.js` | 精算計算ロジック（純関数・自動テスト対象） |
| `data.js` | 共有データ：集合場所・ガソリン価格（`[AUTO-...]`行は自動更新される） |
| `key.js` | チーム共有のNAVITIME APIキー（下記参照） |
| `scripts/update_gas_price.py` | 週次ガソリン価格の自動取得（oil-stat.comから京都府平均） |
| `scripts/validate_data.js` | data.js / key.js の検証（CIで実行） |
| `tests/test_calc.js` | 計算式の自動テスト（CIで実行） |
| `.github/workflows/ci.yml` | push毎にテスト・検証を実行 |
| `.github/workflows/update-gas-price.yml` | 水・木・金 21:30 JSTに価格を自動更新 |

## 外部サービス（すべて無料）

- **NAVITIME Route(car)**（RapidAPI経由）：ルート・高速料金。月500回ハードリミット・カード未登録のため課金は発生し得ない。ETC普通車料金は `fare.unit_1025` 系を優先、なければ `unit_1024`（現金）
- **NAVITIME Spot**（RapidAPI経由）：施設名検索の主軸。Route(car)と同じキーで使えるが**別APIなのでRapidAPIでの無料プランSubscribeも別途必要**（月500回・Route(car)とは別枠のハードリミット）。未契約・枠切れでも検索は下の2つで動き続ける
- **国土地理院**：住所検索・地図タイル
- **OpenStreetMap Nominatim**：施設名検索（フォールバック）
- **GitHub Pages / Actions**：公開・自動化

## APIキーの仕組み

- `key.js` の `APP_SHARED_KEY` にチーム共有キーを設定 → 保護者は設定不要で自動取得が使える
- 公開リポジトリのため**キーは公開**になるが、カード未登録＋500回ハードリミットのため金銭リスクなし。悪用されたら [RapidAPI](https://rapidapi.com/navitimejapan-navitimejapan/api/navitime-route-car) でキーを再発行し `key.js` を差し替える
- 個人キーをアプリの設定タブで登録すると共有キーより優先される（枠を分けたい人向け）
- 共有枠を使い切った月は自動取得が止まる（手入力での計算は常に可能）

## 数字の正しさを担保するしくみ

- **計算式の自動テスト**：push毎にCIが期待値と照合（1円ずれても失敗）
- **共有データの検証**：data.js / key.jsの構文エラー・異常値・全角混入をCIが検出
- **価格急変の停止装置**:自動取得値が前週から15円超変動、または21日超古い、または同日付に複数価格 → 更新を停止して通知
- **画面警告**：価格・距離・高速代・人数が通常範囲外、価格データが10日超古い場合に警告表示
- **誤精算の防止**：会場切替時に距離・料金をクリア／連続タップの応答競合は世代管理で破棄
- **data.js破損時**は計算を停止してエラー表示

## 管理者向けメモ

- **共有データの変更**：GitHubで `data.js`（集合場所）や `key.js`（共有キー）を直接編集 → Commit → 数分で全員に反映
- **ガソリン価格の自動更新が失敗した場合**：GitHubからメールが届く。`scripts/update_gas_price.py` の取得元ページ構造変更が主因になりうる。手動更新は `data.js` の `[AUTO-GAS-PRICE]` 行を直接編集
- **注意**：GitHub Actionsのボットによるpushでは Pages が自動再ビルドされないため、ワークフロー内で `gh api` により明示的にビルド要求している（update-gas-price.yml参照）
- 60日間コミットがないとGitHubが定期実行を自動停止する仕様がある（価格はほぼ毎週変わるため通常は問題なし。停止時はアプリの鮮度警告が検知）

## 免責事項

本アプリは商用サービスではなく、保護者が無償で運営する**チーム内の精算補助ツール**です。

- 金額はチームルールに基づく自動計算です。気になるときは「地図で距離を照合」リンク・LINEコピー文の計算根拠で確認できます（距離・料金は手入力でも調整可能）
- 外部サービス（NAVITIME等）の仕様変更・無料枠の制限により、予告なく自動取得が使えなくなることがあります（その場合も手入力で計算できます）
- 無償運営のため、利用にともなう不利益の補償はいたしかねます
